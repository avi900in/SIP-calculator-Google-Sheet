How to use this:
Open a Google Sheet.

Go to Extensions > Apps Script.

Paste the code from sip_calculator.gs into the editor.

Save and click Run on onOpen (or refresh your spreadsheet).

A new menu SIP Tools will appear in your sheet. Click SIP Tools > Create Calculator.

build a google spreadsheet code for taking following inputs: stock ticker from google finance, start date, tenor period and end date. code will assume $1000 is routinely invested in SIP every period from the start date in the ticker - and will calculate the following - net investment amount, net value of investment today
